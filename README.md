## 🎬 CinaMaxReview
**A full-stack movie review platform with authentication, cloud storage, and AI-assisted reviews.**

🔗 [Live Demo](https://cinamaxreview.netlify.app) · [GitHub](https://github.com/Je9017962/CinaMaxReview-)

**Tech Stack:** React, Vite, React Router, Firebase Auth, Firestore, Netlify, Claude API

**What it does:**
- Browse movies, read and submit star-rated reviews
- Full user auth (sign up, sign in, sign out)
- Edit your profile, display name, bio, and avatar
- AI-assisted review suggestions via Claude API
- Guest path for unauthenticated users
- Light/dark mode, fully responsive

**What I'd do differently:**
Integrate a real movie API (like TMDB) instead of a seeded dataset. 
I'd also update display names retroactively on old reviews rather 
than denormalizing at write time, and add filtering and sorting by 
genre, year, and rating.

🔗 [Live Demo](https://wyleymedia.netlify.app) · [GitHub](https://github.com/Je9017962/WyleyMedia)
